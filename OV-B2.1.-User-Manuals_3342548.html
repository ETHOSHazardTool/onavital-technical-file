User Manual Access
User manual for Users (not Supervisors nor Organizations) can be accessed through the QR code on the packaging sleeve.
The QR code point to the following website Onavital User Manuals | Onalabs where the User Manual can be accessed in pdf format. User Manuals will be available in English and Spanish.

  • OV B2.1.1 Instructions for Use - Users
  • OV B2.1.2 Instructions for Use - Supervisors
  • OV B2.1.3 Instructions for Use - Organizations
