1. Purpose

To document baseline product functionality.

2. Baseline Functionality

  1. Pulse Rate, Oxygen Saturation, Arterial Blood Pressure and Skin Temperature Monitoring: ONAVITAL use optical sensors to continuously monitor your cardiorespiratory health.

  2. Safety and Performance: ONAVITAL complies as a Class IIa PORTABLE TRANSIT-OPERABLE medical device.

  3. Comfortable Bracelet Format: ONAVITAL is a comfortable bracelet so it can be used anywher anytime.

  4. Android and IOS Compatible: ONAVITAL is compatible with Android and IOS.

  5. Cloud Base Data Management: ONAVITAL allows Healthcare Providers to easily acces data collected from ONAVITAL in real time.

  6. Interoperability: ONAVITAL allows interoperability through APIs.
